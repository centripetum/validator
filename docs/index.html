after

after(testValue: Maybe<Date>, value: Maybe<Date>): Either<Maybe<Date>>
Parameters
testValue (Maybe<Date> = END_OF_TIME) The date against which to test
value (Maybe<Date>) The date to be tested
Returns
Either<Maybe<Date>>: Either a Right(Just(value)) if the date value is after the testValue or Left({ errorType: NOT_AFTER_DATE_TIME, testValue, value }) if not
